LC-3b Microarchitecture Simulator

A cycle-level simulator for the LC-3b microarchitecture using C and LC-3b assembly.

  • Implemented virtual memory management with page table and page fault handling.
  • Added support for unaligned access, protection, page fault, and unknown opcode exceptions.
  • Modified the datapath, state diagram, and microsequencer to support pipelining and memory management.
C LC-3b Assembly Computer Architecture
×

4-bit SRAM Design and Characterization

Designed a full custom 4-bit SRAM cell at the transistor level using Cadence Virtuoso: schematic, layout, and parasitic extraction.

  • Characterized propagation delay with HSPICE simulations across varying input slews and output loads.
  • Modeled a 32x32 memory array using RC pi delay model and Spectre simulations to determine worst-case access times.
  • Optimized layout for minimal area while maintaining stable read/write performance.
VLSI Cadence Virtuoso Spectre HSPICE
×

CNNs for Bird Classification

This project focused on training and comparing different convolutional neural networks (CNNs) that classify bird species from images. The goal was to train various models for prediction, test its robustness against image noise, and improve its speed through pruning without sacrificing performance.

  • Dataset: NABirds Dataset from the Cornell Lab of Ornithology, which contains over 48,000 annotated images across 550 categories, including different species, genders, and age groups.
  • Models: ResNet (18, 50, and 101), MobileNetV2, EfficientNet-B0, and DenseNet121
Python CNNs Model Pruning Gaussian Blur
