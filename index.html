LC-3b Microarchitecture Simulator

Developed a pipelined cycle simulator for the LC-3b microarchitecture, implementing datapath digital logic, instruction execution, and exception handling in C and LC-3b assembly.

  • Created microcode and designed a state-cycle simulator to model microsequencer-driven instruction execution.
  • Implemented arithmetic and logic operations, branching, and trap routines according to the LC-3b ISA.
  • Added support for unaligned access, protection, page fault, and unknown opcode exceptions.
  • Extended the datapath to support virtual memory using a one-level page table with address translation and fault detection.
C LC-3b Assembly Microarchitecture Virtual Memory Computer Architecture

CNNs for Bird Classification

This project focused on designing, training, and evaluating different convolutional neural networks (CNNs) to classify bird species from images. The objective was to optimize model prediction accuracy, assess robustness under image noise, and improve speed through model pruning without compromising performance.

  • Dataset: NABirds Dataset from the Cornell Lab of Ornithology with over 48,000 annotated images spanning 550 categories, including species, genders, and age groups.
  • Models: ResNet (18, 50, 101), MobileNetV2, EfficientNet-B0, and DenseNet121
